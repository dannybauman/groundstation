What this is

A console for looking at Earth, right now. Type any place and it pulls together the freshest satellite images, wildfires and disaster alerts, and the week's weather, all on one map. No downloads, no GIS software, no account.

Why it exists

Everything here runs on the same open building blocks Development Seed builds for space agencies and environmental organizations. The point of the prototype: those building blocks are now easy enough to use that an AI agent, or this simple page, can answer real questions about a place in seconds. The same tools behind this console plug into Claude and other AI assistants.

How to use it

Type a place in the top bar and hit Scan, or pan the map anywhere and hit "scan this view." Click a scene to stream it onto the map, click an event to fly to it and load the imagery covering that exact spot. "Generate Earth brief" writes a short situation report for the area, and "Ask the agent" lets you ask a free question, an AI answers using these same tools and hands back a map.

This page is the smallest part

The console is one of three doors into the same engine built for this prototype: a set of tools any AI assistant can use directly (an MCP server plus a Claude skill, so you can ask Claude itself to search imagery, run vegetation math, and produce maps), and an Earth briefing engine that runs on a schedule with nobody asking, checks every place on its watch list, measures what changed since last month, and writes a triaged morning report. Ten live checks against the real data services keep the whole thing honest. What you're clicking is the thin surface over that.

Built on

Imagery search via STAC (Earth Search, NASA VEDA, Microsoft Planetary Computer), tiles and pixel math via TiTiler (Vincent Sarago and team), deep exploration via stac-map (Pete Gadomski) rendering scenes in your browser with deck.gl-raster (Kyle Barron). Events from NASA EONET and GDACS, weather from Open-Meteo. Ideas it follows through on: Will Rynearson's "what if Earth could tell us what we needed to know," and the team's push to make Earth data AI-ready.
